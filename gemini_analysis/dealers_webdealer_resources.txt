## Product Requirements Document: TxDMV webDEALER | eTAG Resources Page

**1.  Introduction**

This document outlines the content requirements for recreating the "webDEALER | eTAG Resources" page on the TxDMV.gov website.  It focuses solely on the text-based content within the main content area of the page, excluding design elements, navigation, search, and footer. The goal is to provide sufficient detail for an AI implementor to accurately reproduce the information presented on the page.

**2.  Document Scope**

This document covers the following:

*   Page Title
*   Main Content Area

**3.  Page Title**

*   **Text:** webDEALER | eTAG Resources

**4.  Main Content Area**

**4.1 Introduction Text**

*   **Image:**  A webDEALER logo is positioned to the right of the introduction text. The logo file name is `/images/content-dealers/webDealer-logo-500x173.png`
*   **Text:**
    **The eTAG System has moved to webDEALER! -** webDEALER is now the system used to create Buyer Tags (or other tags) and process title applications. Users who have accounts for **both** eTAG and webDEALER can now issue tags and create title applications within the same system.
    Individuals who only use eTAG or webDEALER will have the same access in the new system as they do today.
    All users will be able to login to webDEALER using their current eTAG username and password.
    In addition, all paper tags will have enhanced security features, and dealers will be able to pay for Buyer Tags issued to out of state purchasers online by credit card.
